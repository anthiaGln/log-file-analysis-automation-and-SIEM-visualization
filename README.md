# log-file-analysis-automation-and-SIEM-visualization

🔍 Linux Log Analysis for Suspicious Activity

📌 Overview

This project analyzes Linux system logs to identify suspicious login activity such as:

- Failed login attempts

- Invalid/unknown users

- Potential brute-force attacks

The dataset is based on simulated real-world logs from LogHub.

🎯 Objectives

Detect repeated failed login attempts

Identify suspicious IP addresses

Automate log analysis using Python

Export findings into a CSV file
